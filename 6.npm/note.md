## npm 的使用方式
- node package manger

## 全局模块(在命令行的任何地方可以使用) 局部模块
- 为什么全局模块可以直接在任何地方使用

## 全局的模块
- 必须使用package.json 中配置bin参数
- #! /usr/bin/env node

## npm link
就是将当前的目录临时的放到全局下 放标调试


## 本地安装  (--save -S / --save-dev -D)
- npm install webpack --save-dev
- --save-dev 代表的是开要使用
- --save 是开发上线时都需要
- npm uninstall webpack
